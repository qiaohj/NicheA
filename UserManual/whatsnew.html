What's new

Release: 3.0.11 [14 Sep, 2015]

  • Based on comments from two excellent reviewers of Ecography, we did the following modifications in this version.
    • Re-organized the menus to make them clearer via grouping all functions into 6 groups under "Toolbox" menu. Added a new menu 'All functions' before 'About NicheA' which is the same as the original Toolbox menu.
    • Rewrote the 'Export to ENM' function.
    • Expanded N from 2/3D only to 2-N dimensions. The related function supported high dimensions now. But NicheA can only show N in 2/3D in the E space.

Release: 3.0.10 [21 July, 2015]

  • Removed "Show/Hide Points/MVE/CP" menus under "Control" menu. Added "Set the visibility of N's components" under "N(s) list". Now, users can control the visibilities of points, MVE and CP for each N individually.

Release: 3.0.9 [7 July, 2015]

  • Improved "Principal Component Analysis" function.
    • Replaced the internal Java code with a R script to execute PCA to make a better capability. Thank Marcelo Fuentes Hurtado to point out this issue.
    • Provided the ability to transfer the result of PCA from one group of environmental layers to another. Thank Abdallah Samy to give us this requirement.
  • Added a submenu "Change scenario's background" under "Control" menu to switch the background of scenario between black and white.
  • Added a submenu "Check for update" under "About NicheA" menu to compare the working version with the latest online version.

Release: 3.0.8 [24 June, 2015]

  • User can control the MVE/CP/Points for every N via "N(s) list" menu.
  • Added "Calculate AUC value of partial-area ROC approaches" function.
  • Added "Calculate AIC/BIC value" function.
  • Added "Change the range of ENM's result" function.

Release: 3.0.7 [15 May, 2015]

  • Added "BIOMOD2", "dismo" and "Multivariate kernel density estimation (Blonder, B. et al,. 2014)" supported in function "Export to ENMs".
  • Fixed bugs.
    • Fixed the bug in 'Threshold calculator' which causes NicheA crash on Windows when the raster file is locked by the software outside of NicheA (opened in ArcGIS for instance).
    • Fixed the HTML issue caused by "<" and ">" symbols.

Release: 3.0.6 [11 May, 2015]

  • Enhanced 'Controls' menu.
    • Added a new function, "Show/Hide Axes" to control the axes in the scenario.
    • Split "Reset" function to "Switch to X, Y view", "Switch to X, Z view" and "Switch to Y, Z view", which give a multiple angle pan views to user.

Release: 3.0.5 [4 May, 2015]

  • Solved the HTML Viewer issue in Linux version.
